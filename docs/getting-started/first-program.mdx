# Setting Up Your First Program

This guide will walk you through creating and configuring your first program in ProgramMatrix.

## Prerequisites

Before you begin, ensure you have:
- A ProgramMatrix account
- Admin or Program Manager role
- Basic program information ready

## Step-by-Step Guide

### 1. Create New Program

1. Navigate to Dashboard
2. Click "New Program" button
3. Fill in basic details:
   ```typescript
   interface ProgramBasics {
     name: string;
     description: string;
     startDate: Date;
     endDate: Date;
     budget: number;
     currency: string;
     owner: string;
   }
   ```

### 2. Define Program Structure

#### Milestones
1. Add key milestones
2. Set milestone dates
3. Assign owners
4. Define dependencies

#### Teams
1. Add team members
2. Assign roles
3. Set permissions
4. Configure notifications

## Program Configuration

### Dashboard Setup
1. **Select Widgets**
   - KPI trackers
   - Timeline view
   - Risk matrix
   - Budget overview

2. **Configure Layouts**
   - Arrange widgets
   - Set refresh rates
   - Choose data views

### Program Settings
1. **Notifications**
   - Daily updates
   - Weekly reports
   - Risk alerts
   - Milestone reminders

2. **Access Control**
   - Team permissions
   - Stakeholder access
   - External sharing

## Best Practices

### Program Planning
- Start with clear objectives
- Define measurable outcomes
- Set realistic timelines
- Include buffer time

### Team Organization
- Clear roles and responsibilities
- Communication channels
- Regular check-ins
- Documentation standards

## Common Pitfalls to Avoid

1. **Incomplete Setup**
   - Missing key milestones
   - Undefined roles
   - Incomplete documentation

2. **Poor Planning**
   - Unrealistic timelines
   - Insufficient resources
   - Unclear objectives

3. **Communication Gaps**
   - Limited stakeholder involvement
   - Unclear reporting lines
   - Missing notification settings
